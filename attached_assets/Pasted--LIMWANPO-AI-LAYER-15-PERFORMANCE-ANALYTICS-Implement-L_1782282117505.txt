# REQUIRED METRICS

## 1. Total Trades

```python
total_trades
```

Count of CLOSED positions.

---

## 2. Winning Trades

```python
winning_trades
```

Count where:

```python
realized_pnl > 0
```

---

## 3. Losing Trades

```python
losing_trades
```

Count where:

```python
realized_pnl < 0
```

---

## 4. Win Rate

Formula:

```python
winning_trades / total_trades
```

Output:

```python
0-100 %
```

---

## 5. Gross Profit

Formula:

```python
sum(realized_pnl > 0)
```

---

## 6. Gross Loss

Formula:

```python
abs(sum(realized_pnl < 0))
```

---

## 7. Net Profit

Formula:

```python
gross_profit - gross_loss
```

---

## 8. Average Win

Formula:

```python
mean(realized_pnl > 0)
```

---

## 9. Average Loss

Formula:

```python
mean(realized_pnl < 0)
```

Absolute value.

---

## 10. Profit Factor

Formula:

```python
gross_profit / gross_loss
```

Rules:

```python
if gross_loss == 0:
    return None
```

---

## 11. Expectancy

Formula:

```python
(win_rate * avg_win)
-
(loss_rate * avg_loss)
```

---

## 12. Max Drawdown

Build equity curve from CLOSED positions ordered by:

```python
closed_at
```

Calculate:

```python
peak_equity - trough_equity
```

Return:

```python
max_drawdown_usdc
```
