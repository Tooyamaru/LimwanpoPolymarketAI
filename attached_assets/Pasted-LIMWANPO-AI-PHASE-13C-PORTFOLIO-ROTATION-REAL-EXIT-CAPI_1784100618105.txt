==================================================

Final exit harus terjadi jika:

==================================================
STEP 9 — PNL ACCOUNTING
==================================================

Per lot:

unrealized_pnl =
remaining_quantity × (current_exit_price - entry_price)

realized_pnl =
sum closed_quantity × (exit_price - entry_price) - fees

total_pnl =
realized_pnl + unrealized_pnl

Portfolio:

portfolio_realized_pnl =
sum realized_pnl seluruh position

portfolio_unrealized_pnl =
sum unrealized_pnl OPEN/PARTIAL

cumulative_outcome =
realized + unrealized jika itu memang definisi dashboard

Pastikan dashboard labels jujur:

- Resolution Result = realized only
- Cumulative Outcome = realized + unrealized
atau dokumentasikan definisi existing
