5. DEFINE CORRECT EQUITY AND DRAWDOWN FORMULAS
==================================================

Use explicit definitions.

CURRENT EQUITY:

current_equity =
initial_capital
+ total_realized_pnl
+ total_unrealized_pnl

Do not subtract Coverage from equity.

Coverage is allocated capital, not a realized loss.

AVAILABLE CAPITAL:

available =
initial_capital
+ total_realized_pnl
- open_exposure

ALL-TIME PEAK EQUITY:

peak_equity =
MAX(valid historical equity snapshots)

ALL-TIME DRAWDOWN:

drawdown_amount =
MAX(0, peak_equity - current_equity)

drawdown_percent =
drawdown_amount / peak_equity

DAILY EQUITY:

daily_start_equity =
equity snapshot at configured trading-day reset boundary

daily_drawdown_amount =
MAX(0, daily_start_equity - current_equity)

daily_drawdown_percent =
daily_drawdown_amount / daily_start_equity

SESSION DRAWDOWN:

session_drawdown =
MAX(0, session_peak_equity - current_equity)

Do not calculate drawdown using:

- sum of losing trades only
- gross losses without winning trades
- historical lowest equity
- open exposure
- Coverage
- Available instead of equity
- card-local PnL
- current active condition only
- daily loss plus all-time loss twice
- old Replit database values
- stale cached values
- peak from another portfolio/account
