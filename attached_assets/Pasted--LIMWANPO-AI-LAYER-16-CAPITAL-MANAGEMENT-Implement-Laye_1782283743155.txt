# REQUIRED METRICS

## Daily PnL

Calculate today's realized pnl.

Formula:

```python
sum(realized_pnl)
where closed_at >= today_start
```

---

## Weekly PnL

Formula:

```python
sum(realized_pnl)
where closed_at >= week_start
```

---

## Consecutive Loss Count

Order CLOSED positions by:

```python
closed_at DESC
```

Count losses until first winning trade appears.

Example:

```text
-3
-2
-1
+5
```

Result:

```text
3 consecutive losses
```

---

## Equity Curve

Build cumulative realized pnl.

Example:

```text
0
+2
+5
+1
-3
-5
+4
```

Calculate:

```python
peak_equity
current_equity
drawdown_percent
```

Formula:

```python
(peak - current) / peak * 100
```
