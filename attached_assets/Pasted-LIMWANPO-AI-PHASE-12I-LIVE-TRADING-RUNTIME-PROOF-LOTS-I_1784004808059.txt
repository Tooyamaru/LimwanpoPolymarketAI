STEP 7 — LIVE PNL PROOF
==================================================

For a condition_id with an open position:

Calculate manually:

LONG_YES:
unrealized_pnl =
(current_yes_bid - entry_price) × remaining_quantity

LONG_NO:
no_bid = 1 - yes_ask
unrealized_pnl =
(no_bid - entry_price) × remaining_quantity

Then:

total_pnl =
realized_pnl + unrealized_pnl

Compare:

- manual calculation
- database value
- card summary API
- displayed dashboard PNL

All must match within rounding tolerance.

If price missing:

- PNL should show N/A
- not fake $0.00
